# SoundPlaygroundPy
Python port of the experiments that were started in the .NET project.

## Python Dependencies
 - `imgui[glfw]` (requires `libglfw3` and `libglfw3-dev`)
 - `py_linq`

## Usage
```shell
python3 __init__.py
```
